O blueprint cruzado · sequência canônica

Todos os 23 reels seguem (variando intensidade) esta sequência:

[0-3s]  HOOK (A1-A7) — pattern interrupt ou pergunta retórica
[3-7s]  AUTHORITY STAMP (B1-B5) — instala razão pra ouvir
[7-15s] SAVE-CTA opcional (C1-C2) — só se nicho técnico
[15-30s] SETUP da estrutura (D1-D5) — "vou te mostrar 3 coisas / a história / o método"
[30-60s] PRIMEIRO ITEM + ESCALADA (F1-F5) — entrega valor + sustenta atenção
[60-150s] ITENS RESTANTES — repete padrão com escalada crescente
[final 5s] CLOSING (G1-G4) — fecha promessa OU deixa loop aberto pra próximo reel

O que muda entre criadores não é a sequência — é a COMBINAÇÃO dos componentes. Tay = A2 + B3 + sem C + D2. Liliana = A3 + B1 + C1 + D1. Elias = A4 + B4 + C2 + D3. Hanah = A1/A5 + B5 + sem C + D5.

Insight chave: os 4 estilos coexistem porque cada um casa com tipo diferente de audiência. Não é "qual é o melhor" — é "qual combina com o nicho/produto/persona". Tay = curiosidade intelectual. Hanah = ressonância emocional. Liliana = busca por solução técnica. Elias = aspiração documental.
